# ./zero [dave] Design System

> **Identity Protocol for David Schmidt (Cloud Architect)**
> *Established Nov 2025*

## 1. Core Philosophy
The "Zero Dave" brand reflects a transition from legacy physical infrastructure to modern serverless architecture.
* **Aesthetic:** Dark Mode, Terminal, Executable, Precision.
* **Voice:** Disciplined, Documentarian, "Marathon Mindset."

## 2. Color Palette

| Color Name | Hex Code | RGB | Usage |
| :--- | :--- | :--- | :--- |
| **Terminal Black** | `#141419` | `20, 20, 25` | Primary Backgrounds, Badges |
| **Execution Cyan** | `#00FFFF` | `0, 255, 255` | Accents, Cursors, Logo Blocks |
| **Shell White** | `#FFFFFF` | `255, 255, 255` | Primary Text |
| **Comment Grey** | `#999999` | `153, 153, 153` | Subtext, Metadata |

## 3. Typography
* **Font Family:** Monospace (e.g., `Noto Mono`, `Roboto Mono`, `Consolas`).
* **Styling:** Lowercase preferred for brand marks (`./zero`) to mimic executable scripts.

## 4. Deployment Rules

### GitHub
* **Profile:** Use the `dave-schmidt-dev` README configuration.
* **Repos:** All public repos must include the signature footer:
    ```markdown
    <div align="center"><sub>Built by <a href="[https://github.com/dave-schmidt-dev](https://github.com/dave-schmidt-dev)">./zero [dave]</a></sub></div>
    ```

### LinkedIn
* **Banner:** Use `zerodave_linkedin_banner.png` (Terminal Black with right-aligned logo).
* **Profile Pic:** Human headshot (Do not use logo).
